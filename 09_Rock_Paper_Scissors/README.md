## Project Description
This program is a terminal-based implementation of the classic **Rock, Paper, Scissors** game, pitting a human player against the computer. It was designed to demonstrate clean coding practices, input validation, and the use of Python's standard libraries.

### ðŸš€ Key Features
* **Selective Imports:** Optimized using `from random import choice`.
* **Input Sanitization:** Uses `.lower().strip()` to handle accidental spaces or Uppercase inputs.
* **Exception Handling:** Implements `try...except` and `raise ValueError` to manage invalid game moves without crashing.
* **Human vs CPU:** Automated opponent logic using Python's `random` module.
